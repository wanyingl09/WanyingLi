Feature 3: Forms

We'll never share your email with anyone else.

Observations

  • Use type="email" or type="password" for input validation
  • type="checkbox" for checkboxes
  • .btn-primary to have your button stand out
  • .form-text for text input field
  • disabled attribute to prevent inputs. works on text-fields, buttons, checkboxes, etc
  • Make sure to have label tags so that they can be picked up by assistive technologies
